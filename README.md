# Alexander Barsanti Portfolio

A portfolio website showcasing editing, color grading, VFX work, IMAX workshop experience, and production stills.

## Local Development

To view the site locally, open `index.html` in a web browser, or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

Then visit `http://localhost:8000` in your browser.

## Site Structure

- `index.html` - Home page
- `about.html` - About page
- `editing.html` - Editing portfolio
- `color-grading.html` - Color grading portfolio
- `vfx.html` - VFX portfolio
- `imax-workshop.html` - IMAX workshop page
- `production-stills.html` - Production stills gallery
